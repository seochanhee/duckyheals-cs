# duckyheals-cs

Customer-facing static site for DuckyHeals / DuckyStudy, published via GitHub Pages.

## Pages

- `/` — Landing
- `/duckystudy` — Marketing page for the DuckyStudy app
- `/support` — Support and FAQ (used as App Store Support URL)
- `/privacy` — Privacy Policy (used as App Store Privacy URL)

## Deploy

Push to `main`. GitHub Pages serves from the repository root. Custom domain: `www.duckyheals.com` (see `CNAME`).
